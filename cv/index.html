Course Overview

This course provides a comprehensive introduction to the theory and practice of computer vision. Students will learn how to represent, process, and extract meaningful information from digital images. The curriculum bridges classical image processing techniques with modern deep learning approaches, preparing students for research and industry roles in AI, robotics, and intelligent systems.

Learning Outcomes

  • Differentiate between AI, Computer Vision, Image Processing & Graphics
  • Apply image enhancement, thresholding, and histogram techniques
  • Design and train CNNs for classification tasks
  • Evaluate models using standard metrics and prevent overfitting

Prerequisites & Tools

  • Programming Fundamentals (Python)
  • Basic Linear Algebra & Probability
  • Python, NumPy, OpenCV, PyTorch, Jupyter
  • Git & GitHub for version control

Lab Environment: All practical work will be conducted using Jupyter Notebooks. Submission via IUB LMS. Internet required for CDN-based libraries.

Course Syllabus

✅ Covered Material (Weeks 1-10)

Fundamentals & Applications

AI vs CV, Image Processing vs CV, Computer Graphics vs CV, Real-world applications of computer vision

Image Processing Basics

Image formation, contrast stretching, binarization, global/adaptive thresholding, histogram analysis & equalization

Convolution & Feature Maps

Convolution operation & properties, stride, padding, kernel design, feature map generation, output size calculation formula

Neural Networks & Training

ANN vs CNN rationale, CNN architecture stack, forward pass intuition, conceptual backpropagation, epoch, loss functions (BCE), optimizers (SGD, Adam), metrics (Accuracy, Precision, Recall, F1), overfitting/underfitting, bias-variance tradeoff, train/val/test split

📅 Upcoming Material (Weeks 11-15)

Advanced CNN Architectures

ResNet & skip connections, EfficientNet, batch normalization, dropout, regularization techniques

Object Detection

Sliding window vs modern detectors, YOLO architecture, R-CNN family, Non-Maximum Suppression (NMS), mAP evaluation

Semantic Segmentation

Fully Convolutional Networks (FCN), U-Net, encoder-decoder pipelines, IoU & Dice metrics

Transfer Learning & Vision Transformers

Fine-tuning pretrained models, attention mechanism, ViT architecture, self-supervised learning basics

Final Project & Course Review

End-to-end pipeline development, model deployment basics, project presentations, comprehensive review
