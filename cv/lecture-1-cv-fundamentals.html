1 Introduction to Computer Vision

Computer Vision (CV) is a field of artificial intelligence that enables computers and systems to derive meaningful information from digital images, videos, and other visual inputs, and take actions or make recommendations based on that information.

🎯 Definition:

"The science and technology of machines that see and understand visual content."

Why Study Computer Vision?

🔬 Scientific Goals

  • • Understand biological vision systems
  • • Model human perception
  • • Reconstruct 3D from 2D images
  • • Recognize objects and scenes

🏭 Engineering Goals

  • • Build autonomous systems
  • • Create intelligent cameras
  • • Develop medical imaging tools
  • • Enable human-computer interaction

Real-World Applications

🚗

Autonomous Vehicles

Lane detection, traffic sign recognition, pedestrian detection, obstacle avoidance

🏥

Medical Imaging

Tumor detection, X-ray analysis, MRI segmentation, disease diagnosis

📱

Augmented Reality

Face filters, object tracking, 3D reconstruction, virtual try-ons

🔒

Security & Surveillance

Facial recognition, anomaly detection, crowd monitoring, access control

💡 Key Insight:

Not all AI uses vision (e.g., chess-playing AI), but modern CV heavily relies on AI (especially deep learning). Computer Vision is an application domain of AI, not a separate field.

4 Image Formation & Digital Representation

A digital image is a 2D function f(x, y) where x and y are spatial coordinates, and the amplitude of f at any pair (x, y) is called the intensity or gray level of the image.

Image Function: f(x, y) where 0 ≤ f(x, y) ≤ 255 (for 8-bit grayscale)

Image Sampling & Quantization

📐 Sampling (Spatial)

Converting continuous coordinates (x, y) into discrete pixel positions.

Example: A 512×512 image has 262,144 pixels

🎨 Quantization (Intensity)

Converting continuous intensity values into discrete levels.

Example: 8-bit = 2⁸ = 256 gray levels (0-255)

5 Image Histogram

An image histogram is a graphical representation of the intensity distribution in a digital image. It shows the number of pixels for each intensity value in a given image.

Histogram Function: h(rk) = nk
where rk is the k-th intensity level and nk is the number of pixels with that intensity

6 Contrast Stretching

Contrast stretching (also called normalization) is a technique to improve the contrast in an image by stretching the range of intensity values to span the full available range (0-255).

Contrast Stretching Formula:
s = T(r) = (r - rmin) × 255 / (rmax - rmin)
where r = input pixel, s = output pixel, rmin/rmax = min/max intensity in image

Python Implementation

import numpy as np import cv2 def contrast_stretching(image): """ Apply contrast stretching to grayscale image """ # Find min and max intensity values r_min = np.min(image) r_max = np.max(image) # Apply contrast stretching formula stretched = (image - r_min) * 255 / (r_max - r_min) # Convert to uint8 stretched = np.uint8(stretched) return stretched # Example usage img = cv2.imread('low_contrast.jpg', cv2.IMREAD_GRAYSCALE) enhanced = contrast_stretching(img) cv2.imwrite('enhanced.jpg', enhanced)

✅ When to Use Contrast Stretching:

  • • Images with poor contrast (narrow intensity range)
  • • Medical images (X-rays, MRI) for better visibility
  • • Satellite imagery for feature enhancement
  • • Preprocessing step before other operations

7 Thresholding & Binarization

Thresholding is the simplest method of image segmentation. It creates a binary image (black and white) from a grayscale image by setting a threshold value.

Binary Thresholding:
g(x,y) = { 255 if f(x,y) ≥ T
          0 if f(x,y) < T }
where T = threshold value, f = input image, g = output binary image

Types of Thresholding

1. Global Thresholding

Single threshold value for entire image

T = 127 (fixed)
if pixel ≥ 127 → 255
else → 0

2. Adaptive Thresholding

Different threshold for each pixel based on neighborhood

T(x,y) = mean of
neighborhood - C
(C = constant)

Otsu's Automatic Thresholding

Otsu's method automatically finds the optimal threshold value by minimizing intra-class variance (variance within foreground and background). 

import cv2
import numpy as np

# Method 1: Simple Global Thresholding
img = cv2.imread('image.jpg', cv2.IMREAD_GRAYSCALE)
_, thresh1 = cv2.threshold(img, 127, 255, cv2.THRESH_BINARY)

# Method 2: Otsu's Automatic Thresholding
_, thresh2 = cv2.threshold(img, 0, 255, 
                           cv2.THRESH_BINARY + cv2.THRESH_OTSU)
print(f"Otsu threshold value: {thresh2}")

# Method 3: Adaptive Thresholding
thresh3 = cv2.adaptiveThreshold(img, 255,
    cv2.ADAPTIVE_THRESH_GAUSSIAN_C,
    cv2.THRESH_BINARY, 11, 2)

8 Complete Practical Example

Let's combine all techniques in a complete image processing pipeline: 

import cv2
import numpy as np
import matplotlib.pyplot as plt

def complete_image_processing_pipeline(image_path):
    """
    Complete pipeline demonstrating:
    1. Image loading and histogram
    2. Contrast stretching
    3. Thresholding
    """
    
    # Step 1: Load image
    img = cv2.imread(image_path, cv2.IMREAD_GRAYSCALE)
    
    if img is None:
        raise ValueError("Image not found!")
    
    print(f"Original image shape: {img.shape}")
    print(f"Original intensity range: [{img.min()}, {img.max()}]")
    
    # Step 2: Calculate and display histogram
    hist = cv2.calcHist([img], [0], None, [256], [0, 256])
    
    # Step 3: Apply contrast stretching
    img_min = img.min()
    img_max = img.max()
    stretched = ((img - img_min) * 255 / (img_max - img_min)).astype(np.uint8)
    
    print(f"After stretching range: [{stretched.min()}, {stretched.max()}]")
    
    # Step 4: Apply Otsu's thresholding
    _, binary = cv2.threshold(stretched, 0, 255, 
                              cv2.THRESH_BINARY + cv2.THRESH_OTSU)
    
    # Step 5: Display results
    fig, axes = plt.subplots(2, 2, figsize=(10, 8))
    
    axes[0, 0].imshow(img, cmap='gray')
    axes[0, 0].set_title('Original Image')
    axes[0, 0].axis('off')
    
    axes[0, 1].plot(hist)
    axes[0, 1].set_title('Histogram')
    axes[0, 1].set_xlim([0, 256])
    
    axes[1, 0].imshow(stretched, cmap='gray')
    axes[1, 0].set_title('Contrast Stretched')
    axes[1, 0].axis('off')
    
    axes[1, 1].imshow(binary, cmap='gray')
    axes[1, 1].set_title('Binary (Otsu)')
    axes[1, 1].axis('off')
    
    plt.tight_layout()
    plt.show()
    
    return img, stretched, binary

# Usage
original, enhanced, binary = complete_image_processing_pipeline('sample.jpg')

  • ✓ Thresholding converts grayscale to binary for segmentation tasks
  • ✓ Otsu's method automatically finds the optimal threshold value
  • ✓ These are fundamental preprocessing steps for advanced CV tasks
